## VS Electron Login Screen

To run this installation follow the steps provided:

```bash
# Clone this repository
git clone https://github.com/craigwheeler/electron-login.git
# Go into the repository
cd electron-login
# Install dependencies
npm install
# Run the app
npm start
```
